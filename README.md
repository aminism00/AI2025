## 📘 Overview
This repository contains all projects, assignments, and implementations developed for the **Artificial Intelligence (AI)** course under the supervision of **Dr. Saeed Khan Kalantari**.  
The course focuses on the core principles of AI and their practical applications using **Python** and **MATLAB**.

---

## 🧠 Topics Covered
- Uninformed and informed search algorithms  
- Knowledge representation and reasoning  
- Machine learning and neural networks  
- Optimization and heuristic methods  
- Probabilistic models and decision-making  
- Intelligent and logic-based agents
